Some Notes from Recent Python Class Exercises

    1. Clone the remote repository into a local working directory
    2. In your user directory, create a directory and name it git
    3. cd into your git directory
    4. In terminal type the following command:
      "git clone https://github.com/yourusername/yourusername.github.com.git"
    5. Change into your newly cloned directory. Let's say you want to make changes to the README file
    6. Type:
      "git add README"
    7. Make your changes in that file and save them
    8. Type: "git commit -a -m "Updated 2/29/12" "
    9. Git will ask you to insert a message. Not inserting a message will abort the commit
    10. Git will confirm your changes
    11. Type: "git push origin master"
